현재 요금제 안내
• 이 플랫폼의 DB(RESTful Table API)는 클라우드 기반으로 운영됩니다.
• 텍스트/메타 데이터는 무제한에 가깝게 저장 가능하며, file_data(Base64 파일 본체)가 주요 용량을 차지합니다.
• 정확한 DB 용량 한도는 Publish 탭 → 플랜 설정에서 확인하세요.

📌 GitHub 연동 안내
  • 레포가 없으면 연결 테스트 클릭 시 자동 생성됩니다

      Personal access tokens → Classic → repo 체크 필수

STEP 2

AI에게 질문하는 방법

🔑 핵심 키워드 입력으로 답변 품질 UP!

질문 위쪽의 키워드 입력창(최대 3개)에 핵심 단어를 먼저 입력하세요.
AI가 해당 키워드 중심으로 답변하고, 자료실에서도 더 관련된 자료를 보여줍니다.

📌 학교폭력 📌 심의위원회 📌 처리절차

💡 이렇게 질문하면 훨씬 좋은 답변을 받아요!

✅ 예시 1

"학교폭력 사건이 접수되었을 때 교감으로서 처음 해야 할 절차와, 학교폭력대책심의위원회 개최 요건, 피해학생 보호 조치 관련 필요한 서류가 무엇인지 알려주세요. 답변 내용을 바탕으로 보고 기안문 초안과 첨부할 심의 개최 계획서도 작성해 주세요."

✅ 예시 2

"신규 교원이 3월에 부임했을 때 교감으로서 해야 할 학교 적응 지원 업무(멘토링, 연수, 첫 수업 참관 등)와 관련 공문 처리 방법을 단계별로 알려주세요. 기안문 양식과 함께 신규 교사 적응 지원 계획서 초안도 만들어 주세요."

✅ 예시 3

"2학기 학교스포츠클럽 리그 운영 계획을 세우려고 합니다. 예산 편성, 강사 선발, 학생 참가 신청, 결과 보고 절차를 순서대로 알려주세요. 전체 내용을 담은 공문 기안문과 운영 계획서 초안도 함께 작성해 주세요."

✅ 예시 4

"영재학급 학생 선발 과정에서 학부모가 선발 기준에 이의를 제기했을 때 교감으로서 처리해야 할 대응 절차와 유의 사항을 알려주세요. 학부모 이의 처리 결과 보고 기안문과 회신 공문 초안도 함께 만들어 주세요."

💬 질문 끝에 항상 추가해 보세요!

"답변에 대한 기안문과 첨부 붙임문서로 쓸 계획서 초안을 만들어줘"

→ AI가 실제 업무에 바로 활용할 수 있는 공문서 초안까지 함께 작성해 드립니다!
