🏫1. AFTER 소개 및 시작

1.1 AFTER란?

AFTER는 경상남도 중·고등학교 교감 선생님의 행정 업무를 AI가 즉시 도와드리는 웹 시스템입니다. 복무·인사·교육과정·서식 등 1,800개 이상의 교육행정 서식과 Google Gemini AI 질의응답을 제공합니다.

🤖 AI 질문/답변

자연어로 질문하면 즉시 답변 + 관련 서식 다운로드 버튼 제공

📁 서식 자료실

8개 부서별 HWP·PDF·XLSX 등 1,800+ 서식 즉시 다운로드

💬 대화 이어가기

AI와 연속 대화, 파일 첨부 분석 지원

👥 커뮤니티

게시판, 쪽지, 1:1 채팅으로 동료와 소통

1.2 접속 방법

🔗 배포 URL

관리자로부터 안내받은 vip.gensparksite.com 주소로 접속하세요.

주소가 길기 때문에 브라우저 북마크(즐겨찾기)에 저장해두면 편리합니다.

1.3 권장 환경

항목권장 사양

기기PC, 노트북 권장 / 태블릿·모바일도 지원

🔐2. 회원가입 및 로그인

2.1 회원가입

1

우측 상단 "회원가입" 버튼 클릭

로그인 버튼 옆에 "회원가입" 버튼이 표시됩니다.

2

필수 정보 입력

  • 아이디 (영문·숫자, 4~20자)
  • 비밀번호 (8자 이상, 영문+숫자+특수문자 조합)
  • 이름 (실명)
  • 이메일
  • 학교명
  • 개인정보 수집·이용 동의 체크
3

"회원가입 신청" 버튼 클릭

✅ "회원가입 신청이 완료되었습니다. 관리자 승인 후 로그인 가능합니다." 메시지 확인

4

관리자 승인 대기

승인 완료 후 로그인이 가능합니다. 승인 여부는 관리자에게 직접 문의하세요.

⚠️ 가입 전 주의

이미 사용 중인 아이디로는 가입할 수 없습니다. 가입 후 관리자 승인이 완료되기 전까지는 로그인이 불가능합니다.

2.2 로그인

1

우측 상단 "로그인" 버튼 클릭

2

아이디·비밀번호 입력 후 "로그인" 클릭 또는 Enter

3

로그인 완료

상단 버튼이 아이디·포인트 배지로 바뀌고 모든 기능 이용 가능

💡 자동 로그인

로그인 상태는 브라우저를 닫아도 유지됩니다. 다른 사람과 공용 PC를 사용한다면 이용 후 반드시 로그아웃하세요.

2.3 로그아웃 / 비밀번호 변경

  • 로그아웃: 상단 아이디 버튼 클릭 → 프로필 메뉴 → "로그아웃"
  • 비밀번호 변경: 프로필 메뉴 → "비밀번호 변경"
  • 비밀번호 분실: 관리자에게 초기화 요청

🤖3. AI에게 질문하기

3.1 질문 방법

1

사이드바에서 질문할 부서 선택

교감업무 / 교무부 / 과정·평가부 / 연구부 / 인성부 / 정보부 / 과학영재수학환경부 / 체육·안전부

2

질문 입력창에 질문 입력

예: "복무 관련 서식 있어요?", "교원 연수 계획서 작성 방법 알려주세요"

3

"AI에게 물어보기" 버튼 클릭

AI가 답변 생성 중임을 표시합니다. (보통 3~10초)

4

답변 확인

마크다운 형식의 답변 + 관련 서식 다운로드 버튼이 표시됩니다.

⚠️ 포인트 차감

AI 질문 1회당 −5 P 차감됩니다. VIP 회원은 무제한 무료입니다.

3.2 검색 모드

모드검색 범위속도추천 상황
빠른 검색선택 부서만⚡ 빠름특정 부서 서식이 명확할 때
정밀 검색전체 8개 부서🔍 느림어느 부서인지 모를 때

3.3 답변 활용

  • 📥 다운로드 버튼: 답변에 포함된 서식 즉시 다운로드
  • 📋 복사 버튼: 답변 전체 텍스트 클립보드 복사
  • 👍 / 👎 피드백: AI 답변 품질 평가 (향상에 활용)
💡 더 정확한 답변을 받으려면
  • ❌ "서식 있어?" → ✅ "교원 복무 관리 대장 서식 있어요?"
  • ❌ "알려줘" → ✅ "2026년 교육과정 편성 절차 단계별로 알려주세요"
  • 부서를 정확히 선택하면 더 빠르고 정확한 답변을 받을 수 있습니다.

💬4. 대화 이어가기 · 파일 첨부

4.1 추가 질문 (대화 이어가기)

AI 답변을 받은 후 하단 "추가 질문 · 대화 이어가기" 섹션에서 바로 이어서 질문할 수 있습니다. AI는 이전 대화 전체를 기억합니다.

1

답변 하단으로 스크롤

2

추가 질문 입력 → Enter 또는 전송 버튼

줄바꿈은 Shift+Enter

3

채팅 형식으로 대화 이어짐

💡 관련 파일 버튼 유지

추가 질문을 이어가더라도 이전 답변에 있던 관련 파일 다운로드 버튼은 계속 표시됩니다. 파일을 다시 찾을 필요가 없습니다.

4.2 파일 첨부 분석

PDF, HWP, DOCX, XLSX, PPTX, 이미지 파일을 첨부하면 AI가 파일 내용을 분석하여 답변합니다.

1

입력창 왼쪽 📎 (클립) 버튼 클릭

2

분석할 파일 선택 (크기 제한 없음)

3

파일 관련 질문 입력 → 전송

  • "첨부 파일을 요약해주세요"
  • "이 서식 양식처럼 만들어주세요"
  • "첨부된 공문의 주요 내용이 뭔가요?"
지원 형식최대 크기
PDF, HWP, HWPX, DOCX, DOC, XLSX, XLS, PPTX, PPT20 MB
이미지: JPG, PNG, GIF, BMP, WEBP

📁5. 서식 자료실 이용

5.1 자료실 접속

1

사이드바 → "자료실" 탭 클릭

2

전체 서식 수 표시

처음 접속 시 전체 서식 수만 표시됩니다 (로딩 시간 단축).

3

부서 버튼 클릭

원하는 부서 버튼을 클릭하면 해당 부서 서식만 로드됩니다.

💡 lazy loading 방식

자료실은 처음에 전체 건수만 표시하고, 부서 버튼을 클릭할 때만 해당 부서 파일을 불러옵니다. 이전보다 훨씬 빠르게 원하는 자료를 찾을 수 있습니다.

5.2 검색

상단 검색창에 키워드를 입력하면 서식명·설명에서 해당 키워드를 포함한 파일만 표시됩니다.

  • 예: "복무" → 복무 관련 파일만
  • 예: "출결" → 출결 관련 파일만

5.3 파일 다운로드

1

파일 카드에서 "📥 다운로드" 버튼 클릭

2

브라우저 다운로드 폴더에 파일 저장됨

⚠️ 포인트 차감

파일 다운로드 1회당 −10 P 차감됩니다. VIP 회원은 무제한 무료입니다.

5.4 지원 파일 형식

아이콘형식프로그램
📘HWP, HWPX한글 워드프로세서
📕PDFPDF 뷰어 (브라우저 내장 가능)
📗XLSX, XLSMicrosoft Excel
📄DOCX, DOCMicrosoft Word
📊PPTX, PPTMicrosoft PowerPoint

🕐6. 질문 히스토리 조회

6.1 히스토리 보기

1

사이드바 → "히스토리" 탭 클릭

2

대화 목록 확인

내가 질문한 기록은 질문·답변·파일 버튼이 모두 표시됩니다.

다른 사용자의 기록은 제목(질문)과 요약만 표시됩니다. (개인정보 보호)

🔒 타인 히스토리 표시 정책

다른 사람이 질문한 히스토리는 제목과 부서 정보만 표시됩니다. 답변 내용과 관련 파일 버튼은 본인만 열람할 수 있습니다.

6.2 필터링 및 검색

  • 부서 탭: 상단 부서 버튼 클릭으로 특정 부서 기록만 표시
  • 키워드 검색: 검색창에 키워드 입력으로 관련 기록만 표시

6.3 이전 대화 다시 불러오기

1

히스토리 카드의 "🔄 다시 질문" 버튼 클릭

2

원본 답변 화면으로 이동

질문, AI 답변, 관련 파일 다운로드 버튼이 모두 복원됩니다.

3

필요 시 대화 이어가기 가능

💡 히스토리 저장 기간

최근 100건의 대화가 저장됩니다. 100건을 초과하면 가장 오래된 기록부터 자동 삭제됩니다.

6.4 피드백

답변 하단의 👍 좋아요 또는 👎 별로예요 버튼으로 AI 답변 품질을 평가해주세요. 더 나은 답변 개선에 활용됩니다.

📋7. 게시판 이용

7.1 게시판 접속

사이드바 → "게시판" 탭 클릭

7.2 게시글 검색

게시판 상단 검색창에서 제목, 내용, 관리자 답변 텍스트를 통합 검색할 수 있습니다.

  • 검색어는 목록에서 노란색으로 하이라이트됩니다.
  • 관리자 답변에서 검색어가 일치하면 목록에 파란색 미리보기로 표시됩니다.
  • 검색 결과 건수가 상단에 표시됩니다.

7.3 게시글 읽기

  • 게시글 제목 클릭 → 전체 내용 표시
  • 💬 일반 💡 제안 ❓ 문의 📂 자료요청 🚨 파일오류신고
  • 🔒 비밀글 비밀글은 작성자 본인과 관리자만 열람 가능

7.4 게시글 작성

1

우측 상단 "✏️ 글쓰기" 버튼 클릭 (로그인 필요)

2

카테고리 선택

  • 💬 일반: 자유로운 의견
  • 💡 제안: 시스템 개선 제안
  • ❓ 문의: 사용 방법 문의
  • 📂 자료 요청: 필요한 서식 요청
  • 🚨 파일 오류 신고: 파일 오류·시스템 버그
3

제목·내용 입력

4

(선택) 비밀글 체크

🔒 비밀글로 작성하면 제목은 공개되지만 내용은 작성자·관리자만 열람 가능합니다.

5

"등록" 버튼 클릭

7.5 관리자 답변 확인

관리자가 답변한 게시글은 목록에 💬 답변완료 배지가 표시됩니다. 게시글을 클릭하면 파란색 박스로 관리자 공식 답변을 확인할 수 있습니다.

7.6 비밀글 정책

구분일반 사용자작성자 본인관리자
목록에서 제목🔒 아이콘과 함께 표시표시됨표시됨

👥8. 접속 회원 · 쪽지 · 채팅

8.1 접속 회원 목록

홈 화면 우측 패널에 현재 접속 중인 회원 목록이 실시간으로 표시됩니다.

  • 🟢 초록색 점: 최근 3분 이내 활성 상태
  • 👑 왕관 배지: 관리자
  • ⭐ 별 배지: VIP 회원
  • 접속 목록은 1분마다 자동 갱신됩니다.
  • 새로고침 후에도 내 접속 상태가 유지됩니다.

8.2 쪽지 보내기

1

접속 회원 목록에서 쪽지를 보낼 회원 우클릭

2

"쪽지 보내기" 선택

3

내용 입력 (최대 500자) → "전송" 클릭

8.3 받은 쪽지 확인

1

상단 "✉️" 아이콘 클릭

읽지 않은 쪽지가 있으면 빨간 숫자 배지가 표시됩니다.

2

받은 쪽지함 / 보낸 쪽지함 탭 전환

3

쪽지 클릭 → 내용 확인 → 필요 시 "답장" 클릭

8.4 1:1 채팅

1

접속 회원 목록에서 채팅할 회원 우클릭 → "대화하기"

2

채팅창 팝업 → 메시지 입력 → Enter 또는 전송 버튼

💡 채팅 자동 새로고침

채팅창은 8초마다 자동으로 새로운 메시지를 확인합니다.

💰9. 포인트 관리

9.1 포인트란?

AFTER의 주요 기능 이용 시 포인트가 차감됩니다. 포인트가 부족하면 해당 기능이 제한됩니다.

이벤트포인트 변동
회원 승인 시 자동 지급+1,000 P
서식 다운로드−10 P (VIP는 무료)
AI 질문−5 P (VIP는 무료)

9.2 내 포인트 확인

로그인 후 상단 아이디 버튼 옆에 💎 포인트 배지가 표시됩니다.

9.3 포인트 부족 시

⚠️ 포인트 0 이하

포인트가 0이 되면 서식 다운로드와 AI 질문이 제한됩니다. 관리자에게 연락하여 포인트 충전을 요청하세요.

9.4 VIP 등급

관리자가 VIP로 지정하면 포인트 차감 없이 모든 기능을 무제한 사용할 수 있습니다. 프로필에 ⭐ VIP 배지가 표시됩니다.

❓10. 자주 묻는 질문 (FAQ)

Q1. 회원가입 후 로그인이 안 돼요.

A. 회원가입 후 관리자의 승인이 필요합니다. 승인 전까지는 로그인이 제한됩니다. 관리자에게 승인 요청 문의를 하세요.

Q2. AI 답변이 느려요 / 오류가 납니다.

A. Gemini API 사용량 한도(분당 15회)를 초과했을 수 있습니다. 1~2분 후 다시 시도하세요. 계속 오류 발생 시 관리자에게 문의하세요.

Q3. 파일 다운로드가 안 돼요.

  • 포인트 잔액 확인 (다운로드 1회당 −10P)
  • 브라우저 팝업 차단 설정 확인
  • 인터넷 연결 상태 확인
  • 파일 용량이 크면 시간이 걸릴 수 있음

Q4. 게시판 비밀글을 읽을 수 없어요.

A. 비밀글은 작성자 본인과 관리자만 열람할 수 있습니다. 내 글이 아닌 비밀글은 제목만 표시됩니다.

Q5. 히스토리에서 다른 사람 답변이 안 보여요.

A. 개인정보 보호를 위해 타인의 히스토리는 질문 제목만 표시되며, AI 답변과 관련 파일은 본인에게만 표시됩니다.

Q6. 자료실에서 파일이 안 보여요.

A. 자료실은 부서별 클릭 시 로드됩니다. 상단의 부서 버튼을 클릭해보세요. 전체 버튼을 클릭하면 모든 서식이 로드됩니다.

Q7. 접속 회원 목록에 내가 안 보여요.

A. 접속 상태는 1분마다 자동으로 갱신됩니다. 로그인 후 잠시 기다리면 목록에 표시됩니다.

Q8. 비밀번호를 잊어버렸어요.

A. 관리자에게 비밀번호 초기화를 요청하세요.

Q9. 모바일에서도 사용 가능한가요?

A. 네, 모바일 브라우저에서도 사용 가능합니다. 다만 PC·노트북 환경에서 최적화되어 있습니다.

Q10. 포인트가 부족한데 서식이 급히 필요해요.

A. 관리자에게 연락하면 포인트를 즉시 충전받을 수 있습니다. 또는 AI 질문으로 해당 서식이 자료실에 있는지 확인하세요.
